## ✨ Overview

Login pages are the **first impression** of every product — and yet they're the page developers rebuild from scratch the most. This repo fixes that.

**Login Page Templates** is a growing, hand-crafted library of login/signup UI designs spanning every visual language in modern web design — from glassmorphism to brutalism — available in multiple stacks so you can drop one in and ship, not start from zero.

No filler. No half-finished demos. Every template here is **pixel-checked, responsive, accessible, and ready for real integration.**

---

## 🎨 Themes & Designs

| Category | Included Styles |
|---|---|
| **Modern / Trend-driven** | Glassmorphism, Neumorphism, Gradient Mesh, Aurora / Blob Backgrounds |
| **Minimal & Clean** | Flat Minimal, Whitespace-first, Typography-led, Split-screen |
| **Bold & Immersive** | Cyberpunk / Neon, Retro-Wave, 3D Tilt Cards, Animated Particles |
| **Professional / SaaS** | Corporate Dashboard-style, Fintech, Enterprise Sign-in |
| **Dark Mode Native** | True Dark, OLED Black, Dual-theme (light/dark toggle) |
| **Playful** | Illustrated, Animated Mascots, Micro-interaction heavy |

> Each theme includes: **Login, Sign Up, Forgot Password, and OTP Verification** variants.

## 🧩 Tech & Formats

Pick the format that fits your stack — no rewriting required.

- 🟧 **Vanilla HTML / CSS / JS** — zero dependencies, drop into any project
- 🎯 **Tailwind CSS** — utility-first, fully customizable via config
- ⚛️ **React (JSX)** — component-based, state-ready, prop-driven
- 🅱️ **Bootstrap 5** — for rapid prototyping and legacy stacks
- 🌀 **Vue 3** *(rolling out)*

Every template is **self-contained**, mobile-first, and built with clean, readable, well-commented code — designed to be understood in minutes, not hours.

## 🚀 Quick Start

```bash
git clone https://github.com/<your-username>/login-page-templates.git
cd login-page-templates/html-css-js/glassmorphism
open index.html
```

For React/Tailwind builds:

```bash
cd react/glassmorphism
npm install
npm run dev
```
